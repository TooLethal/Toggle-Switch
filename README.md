A simple **HTML + CSS project** that creates a modern toggle switch UI component. The design uses a hidden checkbox input and a styled switch element that slides smoothly when toggled.

## ‚ú® Features
- **Interactive Toggle**  
  - Hidden checkbox (`.toggle-input`) controls the switch state.  
  - Switch (`.toggle-switch`) slides across the track when checked.  

- **Smooth Animation**  
  - CSS `transition` ensures a fluid sliding effect.  
  - Switch moves 190px to the right when activated.  

- **Customizable Styling**  
  - Colors, dimensions, and border radius can be easily adjusted.  
  - Rounded track with circular switch for a clean, modern look.  

- **Responsive Centering**  
  - Flexbox centers the toggle vertically and horizontally on the page.  

- **Minimal Structure**  
  - Simple HTML with `<label>` wrapping the input and switch.  
  - Lightweight CSS for layout, styling, and animation.
